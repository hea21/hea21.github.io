Computer vision for the industry

Manufacturing goods requires extensive monitoring and visual quality inspection for increased production efficiency. 

Automatic monitoring solutions using computer vision enable always-on standardized assessment that reduces costs and detects problems that may not be visible to the human eye.

Our computer vision system provides 
size measurements, segmentation of parts, detection of defects and stains, instance segmentation, and counting.

System architecture

Camera system

Robust to vibration, water and dirt. Includes computer processing,  internet connection, and software controlled focus and zoom.

Deep learning

Image analysis using state-of-the-art deep learning methods. Our methods produce fast and accurate segmentation of objects, detection of detects, and color grading, to name a few examples.

Real-time data processing

Image acquisition and processing in real-time to streamline production.

Real-time vision on conveyor belts

Quality grading and size estimation of moving objects in real-time, to improve defect detection and classification of products.

Multi-perpective imaging of large objects

Suitable for inspection of objects that must be carefully examined from more than one side.
